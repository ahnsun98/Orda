### 활용 데이터, API
- 산 정보 : 공공 데이터 포털  
https://www.data.go.kr/data/15058682/openapi.do
- 산 위치 지도 및 맛집 정보 지도: KAKAO 지도 API  
https://apis.map.kakao.com/
- 산 이미지 크롤링: NAVER API  
https://openapi.naver.com/
- 등산 경로 이미지 크롤링(정적 웹 크롤링)  
https://ilikesan.com/entry/전국-등산지도-모음
